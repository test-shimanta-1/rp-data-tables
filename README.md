# Responsive Data Tables

Responsive Data Tables is a WordPress plugin that helps you manage and display inventor-related information in responsive, searchable, and feature-rich data tables across your website.

All inventor records are managed centrally inside the plugin and can be shown anywhere using shortcodes.

---

## Description

Responsive Data Tables allows administrators to add, edit, and manage multiple inventor records from a single location in the WordPress admin panel. The inventor data can then be displayed on multiple pages or posts using shortcodes in the form of responsive and interactive data tables.

The plugin supports global search, column-wise search, pagination, and export functionality to improve data usability and readability.

---

## Key Features

- Centralized management of inventor information
- Responsive and mobile-friendly data tables
- **Global search** across all table data
- **Column-wise search** for precise filtering
- **Pagination support** to improve readability and performance
- Export table data in:
  - Excel
  - PDF
  - Print format
- Reusable tables using shortcodes
- Clean and user-friendly interface

---

## Use Cases

- Manage inventor information at one central place
- Display inventor data tables on multiple pages or posts
- Search inventor data globally or column-wise
- Export inventor records for reporting

You can store and display the following information:
- Inventor Names
- Project Title
- IPA
- Filing Date
- Grant Number
- Grant Date
- Financial Year
- Calendar Year

## Usage

Use the following shortcode to display the inventor data table on any page or post:

```text
[responsive_data_table]
```
